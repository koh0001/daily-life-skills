# OTT Guide — 작품 검색 & 맞춤 추천

OTT 콘텐츠의 바다에서 "지금 나에게 딱 맞는 한 편"을 찾아주는 스킬이다. 특정 작품이 어디서 서비스되는지 즉시 확인하거나, 남은 시간과 기분에 맞는 작품을 추천받을 수 있다.

---

## 사용 모드

### 모드 1: 작품 검색 (어디서 볼 수 있어?)

사용자가 **특정 작품명**을 언급하면 이 모드로 진입한다. 핵심은 "한국에서 지금 어디서 볼 수 있는가"이다.

**검색 프로세스:**

1. 웹 검색으로 해당 작품의 한국 OTT 서비스 현황을 확인한다
   - 검색어 예시: `"체르노빌" OTT 한국 스트리밍`, `"Chernobyl" justwatch korea`
   - JustWatch, 키노라이츠 등 OTT 통합 검색 사이트를 우선 참조
2. 작품 기본 정보를 함께 수집한다

**작품 검색 카드 포함 정보:**

| 항목 | 설명 |
|------|------|
| 🎬 제목 | 원제 + 한국어 제목 |
| 📺 유형 | 영화 / 드라마 / 애니 / 다큐 |
| ⭐ 평점 | IMDb, 로튼토마토, 왓챠 피디아 (가능한 것만) |
| 🎭 장르 | 장르 태그 (최대 3개) |
| ⏱️ 러닝타임 | 영화: 총 시간 / 시리즈: 시즌 수 × 에피소드 수 × 회당 시간 |
| 📅 연도 | 제작 연도 |
| 🌍 국가 | 제작 국가 |
| 📖 시놉시스 | 스포일러 없는 2~3문장 소개 |

**OTT 서비스 현황 테이블:**

| OTT | 서비스 형태 | 가격 | 비고 |
|-----|-----------|------|------|
| 넷플릭스 | 구독 포함 | 월 5,500~17,000원 | 광고형/스탠다드/프리미엄 |
| 디즈니+ | 구독 포함 | 월 9,900~13,900원 | - |
| 쿠팡플레이 | 구독 포함 | 월 7,890원 (로켓와우) | 와우 멤버십 포함 |
| 웨이브 | 개별 구매 | 편당 1,500원 | - |
| 왓챠 | - | - | 미서비스 |

- 서비스되는 플랫폼이 없는 경우: "현재 한국 OTT에서 서비스되지 않습니다"라고 명시하고, 대안(해외 VPN 필요 여부, DVD/블루레이, 해외 구매 등)을 안내
- 여러 플랫폼에서 서비스될 때: 가장 가성비 좋은 옵션을 표시 (이미 구독 중인 플랫폼을 사용자가 알려주면 그것을 우선)

**딥링크 생성 규칙:**

각 OTT 플랫폼의 작품 페이지로 바로 갈 수 있는 링크를 제공한다:

- **넷플릭스**: `https://www.netflix.com/search?q={작품명}`
- **디즈니+**: `https://www.disneyplus.com/ko-kr/search/{작품명}`
- **쿠팡플레이**: `https://www.coupangplay.com/search?q={작품명}`
- **웨이브**: `https://www.wavve.com/search?searchWord={작품명}`
- **왓챠**: `https://watcha.com/search?query={작품명}`
- **티빙**: `https://www.tving.com/search?keyword={작품명}`
- **애플TV+**: `https://tv.apple.com/kr/search/{작품명}`

검색 URL을 사용하되, 웹 검색에서 직접 링크를 찾으면 그것을 우선 사용한다.

### 모드 2: 맞춤 추천 (뭐 볼까?)

사용자가 "추천해줘", "뭐 볼까", "시간 남는데" 같은 표현을 쓰면 이 모드로 진입한다.

**추천을 위한 조건 파악:**

사용자가 이미 정보를 줬다면 추가 질문 없이 바로 추천한다. 정보가 부족하면 아래 항목만 가볍게 물어본다 (한 번에 3개 이하):

| 조건 | 설명 | 기본값 |
|------|------|--------|
| ⏱️ 가용 시간 | 남은 시간 또는 원하는 러닝타임 | 2시간 |
| 🎭 장르/기분 | 원하는 장르 또는 현재 기분 | 미지정 → 다양하게 |
| 📺 구독 OTT | 현재 구독 중인 플랫폼 | 미지정 → 전체 OTT |
| 🌍 선호 국가 | 한국/미국/일본/영국 등 | 미지정 → 전체 |
| 📊 콘텐츠 유형 | 영화/드라마/애니/다큐 | 미지정 → 전체 |
| 👥 시청 인원 | 혼자/커플/가족/친구 | 혼자 |

**시간 기반 추천 로직:**

사용자가 시간을 명시하면, 해당 시간에 맞는 콘텐츠를 골라야 한다:

| 가용 시간 | 추천 콘텐츠 유형 |
|-----------|-----------------|
| ~30분 | 숏폼, 에피소드 1개 (시트콤, 애니메이션) |
| 30분~1시간 | 드라마 에피소드 1~2개, 단편 다큐 |
| 1~2시간 | 영화 1편, 미니시리즈 2~3편 |
| 2~4시간 | 영화 1~2편, 미니시리즈 정주행 (4~6편) |
| 4~6시간 | 시즌 1 정주행 (8~10편 시리즈), 영화 마라톤 |
| 6시간+ | 장편 시리즈 정주행, 3부작 영화 마라톤 |

**기분 기반 매핑:**

사용자가 기분이나 상황을 설명하면 적절한 장르로 매핑한다:

| 기분/상황 | 추천 장르 |
|-----------|----------|
| 스트레스 풀고 싶어 | 코미디, 힐링, 가벼운 로맨스 |
| 머리 비우고 싶어 | 액션, 코미디, 예능 |
| 생각할 거리가 필요해 | SF, 스릴러, 다큐멘터리 |
| 감성적인 기분 | 드라마, 로맨스, 음악 |
| 심심해 죽겠어 | 미스터리, 스릴러, 범죄 (몰입감 강한) |
| 무서운 거 보고 싶어 | 호러, 심리 스릴러 |
| 가족이랑 같이 | 가족 영화, 애니메이션, 가벼운 코미디 |
| 커플끼리 | 로맨틱 코미디, 감성 드라마, 뮤지컬 |
| 아이랑 같이 | 애니메이션, 키즈, 가족 영화 (연령 등급 확인) |

**추천 개수:**

- 기본 3~5개 추천
- 사용자가 더 원하면 추가 추천
- 각 추천은 반드시 "왜 이 작품인지" 한 줄 이유를 포함

### 모드 3: 시리즈 가이드 (정주행 도우미)

사용자가 시리즈/드라마의 시청 순서, 스핀오프 관계, 정주행 가이드를 요청하면 이 모드로 진입한다.

**시리즈 가이드 포함 정보:**

| 항목 | 설명 |
|------|------|
| 🗺️ 시청 순서 | 공개 순서 vs 스토리 순서 (다를 경우 둘 다 제시) |
| 📊 시즌별 평점 | 시즌별 IMDb/로튼토마토 점수 변화 |
| ⏱️ 총 소요 시간 | 전체 정주행에 필요한 시간 계산 |
| 🔀 스핀오프 맵 | 본편/스핀오프/프리퀄 관계도 |
| ⚡ 핵심 시즌 | "시간 없으면 이 시즌만 보세요" 압축 가이드 |
| 📈 몰입 곡선 | "처음에 좀 느리지만 3화부터 빨라짐" 같은 페이스 안내 |
| ✅ 완결 여부 | 완결/진행중/취소 상태 |

**유니버스/프랜차이즈 시청 순서 예시 패턴:**

```
[MCU, 스타워즈, DC 등 대형 프랜차이즈]
→ 공개 순서 / 타임라인 순서 / 입문자 추천 순서
세 가지 옵션을 제시하고 사용자가 선택하게 한다
```

## 시각화 — React JSX 컴포넌트

모든 추천 결과는 **React JSX 파일(.jsx)**로 생성하여 대화 안에서 바로 렌더링한다. HTML 파일을 별도로 만들지 않는다.

### 출력 방식

1. `.jsx` 파일을 workspace에 저장한다
2. `export default` 함수형 컴포넌트로 작성한다
3. React hooks (`useState`, `useMemo` 등)를 import하여 인터랙티브하게 만든다
4. 스타일은 인라인 스타일 또는 Tailwind 유틸리티 클래스를 사용한다
5. 외부 라이브러리: `recharts`(차트), `lucide-react`(아이콘) 사용 가능

### 작품 카드 컴포넌트 구조

각 작품 카드는 다음 요소를 포함한다:

- **포스터 영역**: CSS gradient 배경 + 대형 이모지 + 제목
- **평점 뱃지**: `#16a34a`(8.0+), `#ca8a04`(6.0~7.9), `#dc2626`(6.0 미만)
- **완결/방영중 뱃지**: 보라(완결) / 파랑(방영중)
- **메타 정보**: 유형, 국가, 연도, 에피소드 수
- **장르 태그**: 둥근 태그 형태
- **시놉시스**: 스포일러 없는 2~3문장
- **추천 이유**: 보라 강조 박스 (`borderLeft: 3px solid #8b5cf6`)
- **러닝타임 바**: 가용 시간 대비 진행률 표시
- **몰입 곡선 텍스트**: 어느 시점에 재미가 붙는지
- **OTT 바로가기 버튼**: 플랫폼 색상별 버튼 + `target="_blank"` 딥링크

### OTT 플랫폼 색상

| 플랫폼 | 색상 |
|--------|------|
| 넷플릭스 | `#E50914` |
| 디즈니+ | `#113CCF` |
| 쿠팡플레이 | `#FF5722` |
| 웨이브 | `#1A1A2E` |
| 왓챠 | `#FF0558` |
| 티빙 | `#FF0143` |
| 애플TV+ | `#000000` |
| 크런치롤 | `#F47521` |

### 인터랙티브 기능 (useState 활용)

- **시간 타임바**: 작품 선택/해제로 하루 시간 배분 시각화
- **필터**: 장르, 완결 여부로 카드 필터링
- **정렬**: 추천순, 평점순, 러닝타임순
- **카드 선택**: 클릭으로 시청 계획에 추가/제거

### 디자인 톤

- 다크 테마 (`background: #0a0a0f`, `color: #e0e0e0`)
- 그라데이션 히어로 섹션 (보라~핑크~주황)
- 카드: 반투명 글래스 효과 (`rgba(255,255,255,0.03)`)
- 모바일 대응: 카드 너비 100%, 터치 영역 44px 이상, 폰트 최소 14px

## 특수 상황 대응

### 구독 최적화 조언

사용자가 "어떤 OTT 구독하면 좋을까?" 같은 질문을 하면:
- 현재 관심 장르/작품을 파악
- 각 OTT별 강점 요약 (한국 기준):

| OTT | 강점 | 월 요금 (기본) |
|-----|------|----------------|
| 넷플릭스 | 오리지널 드라마/영화, 글로벌 콘텐츠 | 5,500~17,000원 |
| 디즈니+ | 마블/스타워즈/픽사, 해외 드라마 | 9,900~13,900원 |
| 쿠팡플레이 | 스포츠(EPL), 한국 예능, 로켓배송 번들 | 7,890원 |
| 웨이브 | 한국 드라마/예능 (지상파 연계) | 7,900~13,900원 |
| 왓챠 | 독립/예술 영화, 클래식 | 7,900~12,900원 |
| 티빙 | 한국 드라마/예능 (CJ 계열), 스포츠 | 7,900~17,000원 |
| 애플TV+ | 고품질 오리지널, 적은 콘텐츠 양 | 13,900원 |

- 사용자 취향에 맞는 상위 2~3개 추천하되, 웹 검색으로 최신 요금을 확인한다

### 시청 기록 기반 추천

사용자가 "최근에 OO 봤는데 비슷한 거 추천해줘"라고 하면:
- 해당 작품의 장르, 분위기, 테마를 분석
- "이 작품을 좋아했다면" 패턴으로 3~5개 추천
- 단순 장르 매칭이 아니라 분위기/톤/주제가 유사한 작품 위주

### 최신/인기작 요청

"요즘 뭐가 인기야?", "넷플 신작 뭐 나왔어?" 같은 질문:
- 반드시 웹 검색으로 최신 정보 확인 (학습 데이터의 인기작 정보는 오래됐을 수 있음)
- 검색어 예시: `넷플릭스 한국 인기작 2026`, `OTT 신작 이번주`
- 각 OTT 공식 톱10이나 인기 차트를 참조

### 연령 등급 안내

가족/아이와 함께 볼 작품을 추천할 때는 반드시 연령 등급을 표시한다:
- 🟢 전체 관람가 / ALL
- 🟡 7세 / 12세 이상
- 🟠 15세 이상
- 🔴 청소년 관람불가 / 19+

## 웹 검색 전략

### 작품 검색 시

1차: `"{작품명}" site:justwatch.com korea` 또는 `"{작품명}" site:kinolights.com`
2차: `"{작품명}" OTT 한국 스트리밍 서비스`
3차: `"{작품명}" netflix korea` / `"{작품명}" 디즈니플러스 한국` (특정 OTT)

### 추천 시

1차: `{장르} OTT 추천 {연도}` / `{OTT플랫폼} 인기작 {연도}`
2차: `{시간}시간 영화 추천` / `{기분} 드라마 추천`
3차: 관련 커뮤니티/리뷰 사이트 (씨네21, 왓챠피디아 등)

### 시리즈 가이드 시

1차: `"{시리즈명}" 시청 순서` / `"{시리즈명}" watch order`
2차: `"{시리즈명}" 스핀오프 관계` / `"{시리즈명}" 프랜차이즈 가이드`

## 응답 원칙

1. **스포일러 절대 금지**: 시놉시스는 1화 이내 내용만, 반전/결말 언급 금지
2. **최신 정보 우선**: OTT 서비스 현황은 수시로 바뀌므로 반드시 웹 검색으로 확인
3. **가용 시간 존중**: 6시간 남았다고 6시간짜리만 추천하지 말 것. 여유 있는 선택지를 제공 (짧은 것 + 딱 맞는 것 + 약간 긴 것)
4. **구독 여부 고려**: 사용자가 구독 OTT를 알려주면 해당 플랫폼 우선, 안 알려주면 가장 접근성 좋은 것부터
5. **추천 이유 필수**: 모든 추천에는 "왜 이 작품인지" 한 줄 근거를 포함한다. 단순히 "인기작이라서"가 아니라 사용자의 조건에 맞는 이유를 제시
6. **다양성 확보**: 추천 리스트에 같은 장르/국가만 나열하지 않는다. 최소 2개 이상의 장르/국가를 섞되, 사용자가 특정 장르를 원하면 그 안에서 다양성 확보
