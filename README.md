# Daily Life Skills

일상생활에서 유용한 Claude 스킬 모음입니다.

각 스킬은 Claude Code / Cowork에서 사용할 수 있는 독립적인 프롬프트 기반 도구로, 특정 작업을 더 체계적이고 풍부하게 수행할 수 있도록 도와줍니다.

## 스킬 목록

| 스킬 | 설명 | 주요 기능 |
|------|------|----------|
| [travel-planner](./travel-planner/) | 맞춤형 여행 계획 수립 | 항공권/숙소 비교, 일정 수립, 예산 관리, 현지 물가 체감 지수 |
| [local-restaurant-finder](./local-restaurant-finder/) | 로컬 맛집 추천 | 지역/음식/예산별 맛집 검색, 인터랙티브 비교 카드, 예약 딥링크 |
| [liquor-guide](./liquor-guide/) | 주류 정보 & 추천 | 맛 프로필 레이더 차트, 면세점/마트 가격 비교, 입문자 추천, 음식 페어링 |

## 설치 방법

### Claude Code에서 사용하기

원하는 스킬 폴더를 프로젝트의 `.claude/skills/` 디렉토리에 복사하면 됩니다:

```bash
# 예: travel-planner 스킬 설치
cp -r travel-planner /your-project/.claude/skills/
```

### Cowork에서 사용하기

`.claude/skills/` 폴더에 스킬을 추가하면 Cowork 세션에서 자동으로 인식됩니다.

## 스킬 상세

### travel-planner

해외 여행 계획을 처음부터 끝까지 한 번에 수립해주는 종합 여행 플래너입니다.

**주요 기능:**
- 여행 성향 파악 후 맞춤 일정 수립
- 항공권 가격 비교 (스카이스캐너/네이버/구글 플라이트) + 예약 딥링크
- 숙소 비교 (아고다/호텔스닷컴/부킹닷컴) + 예약 딥링크
- 현지 물가 체감 지수 (서울 대비 비교 차트)
- 예산 배분 인터랙티브 위젯
- Notion 연동 일정 저장 (MCP 사용 가능 시)
- Google Maps 딥링크 자동 생성

**트리거 예시:**
- "도쿄 여행 계획 짜줘"
- "예산 150만원으로 동남아 어디 갈까"
- "항공권 싼 거 찾아줘"

### local-restaurant-finder

원하는 조건에 맞는 맛집을 찾아주는 스킬입니다. 국내/해외 모두 지원합니다.

**주요 기능:**
- 지역, 음식 종류, 예산, 인원, 분위기 등 조건 기반 검색
- 인터랙티브 HTML 카드 위젯으로 비교 (필터/정렬 기능)
- Google Maps 길찾기 딥링크
- 예약 플랫폼 딥링크 (캐치테이블/네이버 예약/OpenTable 등)
- 예산별 색상 구분 (가성비/보통/고급)
- 특수 상황 대응 (혼밥/데이트/단체 회식)
- 해외: 현지 통화+원화 병기, 주문 팁 포함

**트리거 예시:**
- "강남 맛집 추천해줘"
- "을지로 분위기 좋은 이탈리안"
- "도쿄 시부야 라멘집 추천"
- "8명 회식 장소 찾아줘"

### liquor-guide

해외 리쿼샵이나 면세점에서 낯선 술을 만났을 때, 즉시 전문가 수준의 정보를 제공하는 주류 가이드입니다.

**주요 기능:**
- 즉석 조회 카드 (매장에서 빠른 판단용)
- 맛 프로필 레이더 차트 (위스키/진/럼/보드카/테킬라/와인/사케 각각 6축)
- 가격 비교 (글로벌 시세/면세점/한국 마트) + 구매 판정 (🟢/🟡/🔴)
- 입문자 단계별 추천 루트 (Step 1→2→3)
- 음식 페어링 (베스트 매치/간편 안주/정찬)
- 비교 위젯 (여러 병 나란히 비교)
- 면세 한도 안내 (2병, $400 이내)

**트리거 예시:**
- "이 위스키 어때?"
- "면세점에서 뭐 사면 좋을까"
- "위스키 입문 추천해줘"
- "야마자키랑 히비키 비교해줘"
- "진토닉 만들 진 추천"
