# Liquor Guide — 주류 정보 & 추천 전문가

낯선 술병 앞에서 막막할 때, 어떤 술을 사야 할지 고민될 때, 이 스킬이 즉시 전문가 수준의 정보를 제공한다. 매장에서 빠른 조회부터 집에서 꼼꼼한 비교까지 모두 지원한다.

---

## 사용 모드

### 모드 1: 즉석 조회 (매장용)

사용자가 특정 술 이름을 말하면 바로 핵심 정보를 제공한다. 리쿼샵이나 면세점에서 빠르게 판단할 수 있도록 **간결하고 핵심적인 정보**를 우선 보여준다.

**즉석 조회 카드 포함 정보:**

| 항목 | 설명 |
|------|------|
| 🏷️ 이름 | 정식 명칭 + 한국어 표기 |
| 🏴 원산지 | 국가, 지역, 증류소/와이너리 |
| 🥃 종류 | 대분류 (위스키/럼/진 등) + 세부 분류 (스카치 싱글몰트, 버번 등) |
| 🎂 숙성 | 숙성 연수 (해당 시) |
| 👅 맛 프로필 | 핵심 풍미 3~5개 키워드 + 레이더 차트 |
| 🔥 도수 | ABV (%) |
| 💰 시세 | 글로벌 평균가 + 한국 시세 (면세점/마트) |
| ⭐ 평가 | 주요 평점 (가능한 경우) |
| 🍽️ 페어링 | 어울리는 음식 2~3개 |
| 💡 한줄평 | "이런 사람에게 추천" 한 줄 요약 |

### 모드 2: 비교 검토 (집에서)

여러 병을 비교할 때 인터랙티브 HTML 카드 위젯으로 나란히 비교한다.

## 맛 프로필 시스템

모든 주류의 맛을 일관된 체계로 표현한다. 종류별로 적합한 축을 사용하되, 기본 구조는 동일하다.

### 위스키 맛 축 (6축 레이더 차트)

| 축 | 설명 | 예시 |
|----|------|------|
| 스모키 | 피트/연기향 정도 | 아일라 위스키는 높음 |
| 스위트 | 단맛/바닐라/꿀 | 버번은 대체로 높음 |
| 프루티 | 과일향 정도 | 스페이사이드 계열 |
| 스파이시 | 매콤/후추/시나몬 | 라이 위스키 계열 |
| 바디 | 묵직함/가벼움 | 라이트~풀바디 |
| 피니시 | 여운 길이 | 짧음~매우 김 |

### 기타 주류 맛 축

**럼**: 스위트 / 캐러멜 / 스파이시 / 트로피컬 / 오크 / 바디
**진**: 주니퍼 / 시트러스 / 플로럴 / 허벌 / 스파이시 / 드라이
**보드카**: 클린 / 크리미 / 스위트 / 시트러스 / 그레인 / 스무스
**테킬라**: 아가베 / 시트러스 / 스파이시 / 허벌 / 스모키 / 스위트
**와인**: 바디 / 타닌 / 산도 / 스위트 / 프루티 / 미네랄
**사케**: 아마쿠치(단맛) / 카라쿠치(드라이) / 우마미 / 프루티 / 산도 / 바디

맛 프로필은 0~5점으로 표기하며, 레이더 차트로 시각화한다.

## 가격 정보 & 면세점 비교

`web_search`로 가격 정보를 수집하여 "여기서 사는 게 이득인지" 판단을 도와준다.

### 수집할 가격 데이터

| 채널 | 검색 방법 | 비고 |
|------|----------|------|
| 글로벌 시세 | `"{술이름} price"` 검색 | Wine-Searcher, Master of Malt 등 참고 |
| 한국 면세점 | `"{술이름} 면세점 가격"` 검색 | 신라/롯데/신세계 면세점 |
| 한국 마트/편의점 | `"{술이름} 한국 가격"` 검색 | 이마트/코스트코/편의점 |
| 현지 가격 | 사용자가 말한 가격 또는 현지 웹 검색 | 사용자가 가격을 말하면 비교 기준으로 활용 |

### 가격 판단 가이드

수집한 가격을 비교하여 직관적인 판단 지표를 제공한다:

```
💰 가격 비교 — Macallan 12년
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
현지 리쿼샵:    $55 (약 73,000원)
한국 면세점:    68,000원
한국 마트:      95,000원
글로벌 평균:    $60 (약 80,000원)
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
💡 판정: 면세점보다 약간 비싸지만 마트 대비 23% 저렴
   → 한국에서 구하기 어려운 병이면 구매 추천 ✅
```

**판정 기준:**
- 한국 마트 대비 30% 이상 저렴 → 🟢 "강력 추천! 무조건 사세요"
- 한국 마트 대비 15~30% 저렴 → 🟡 "괜찮은 가격, 사도 좋아요"
- 한국 마트와 비슷하거나 비쌈 → 🔴 "한국에서 사는 게 나을 수 있어요"
- 한국 미입고 제품 → ⭐ "한국에서 구할 수 없는 병! 기회가 되면 구매"

면세 한도(해외 입국 시 주류 2병, 합계 $400 이내)도 함께 안내한다.

## 입문자 추천 가이드

사용자가 "위스키 입문", "럼 처음 마셔보려고" 등 입문 관련 질문을 하면 단계별 추천을 제공한다.

### 추천 흐름

1. **현재 취향 파악** — 평소 좋아하는 맛/음료가 뭔지 가볍게 물어본다
   - 단 음료 좋아함 → 셰리 캐스크 위스키, 스파이스드 럼
   - 커피/쓴 맛 좋아함 → 피티 위스키, 네그로니용 진
   - 과일 좋아함 → 과일향 진, 프루티 사케
   - 잘 모르겠음 → 가장 대중적인 입문 병부터 추천

2. **단계별 추천** — 가벼운 것에서 깊은 것으로 3단계 제안
   ```
   Step 1 (입문) → Step 2 (탐험) → Step 3 (심화)
   ```

3. **각 추천에 포함할 정보:**
   - 왜 이 단계에서 이 술인지 (예: "바닐라 단맛이 강해서 입문자가 편하게 즐길 수 있음")
   - 추천 음용법 (니트/온더록/하이볼/칵테일)
   - 예상 가격대

### 주류별 입문 루트 예시 (참고용, 웹 검색으로 최신 추천 확인)

**위스키 입문:**
- Step 1: 글렌피딕 12년 / 버팔로 트레이스 (부드럽고 접근성 좋음)
- Step 2: 맥캘란 12년 / 와일드터키 101 (풍미 확장)
- Step 3: 라프로익 10년 / 부커스 (개성 있는 맛 탐험)

**진 입문:**
- Step 1: 헨드릭스 / 탱커레이 (균형 잡힌 플로럴/클래식)
- Step 2: 몽키47 / 로쿠 (복합적 보태니컬)
- Step 3: 아일라 진 / 네이비 스트렝스 (강렬한 개성)

## 음식 페어링

추천한 술에 어울리는 음식을 함께 안내한다.

### 페어링 원칙

| 원칙 | 설명 | 예시 |
|------|------|------|
| 보완 | 비슷한 풍미끼리 매칭 | 스모키 위스키 + 훈제 연어 |
| 대비 | 반대 성격으로 균형 | 달콤한 럼 + 짭짤한 치즈 |
| 강도 매칭 | 술의 무게감과 음식 무게감 맞추기 | 라이트 바디 진 + 가벼운 샐러드 |
| 지역 매칭 | 같은 지역의 술과 음식 | 사케 + 사시미, 테킬라 + 타코 |

### 페어링 제안 형식

각 술에 대해 3가지 페어링을 제안한다:
- 🥇 **베스트 매치** — 가장 추천하는 조합 + 이유
- 🍿 **간편 안주** — 편의점/마트에서 쉽게 구할 수 있는 것
- 🍽️ **정찬 페어링** — 레스토랑/특별한 날에 어울리는 조합

## 시각화 — 인터랙티브 카드 위젯

### 단일 조회 카드

한 병을 조회할 때 보여주는 카드:

**카드 구성:**
- 헤더: 술 이름 + 종류 배지 + 원산지 플래그
- 맛 프로필: 6축 레이더 차트 (SVG 또는 CSS로 구현)
- 핵심 정보 그리드: 도수, 숙성, 용량
- 가격 비교 바: 현지/면세점/마트 가격을 수평 바로 비교 (최저가 강조)
- 페어링 칩: 추천 음식 3개를 칩으로 표시
- 한줄평: 추천 대상 한 줄 요약
- 구매 판정 배지: 🟢/🟡/🔴 + 판정 이유

### 비교 위젯

여러 병을 비교할 때:

**위젯 구성:**
- 필터/정렬: 가격순, 평점순, 도수순, 종류별 필터
- 카드 나란히 배치 (2~3열)
- 레이더 차트 오버레이: 여러 술의 맛 프로필을 한 차트에 겹쳐 표시
- 가격 비교 테이블: 채널별 가격을 한눈에 비교
- 하이라이트: "가성비 최고", "입문자 추천", "선물용 추천" 등 배지

### 입문자 로드맵 위젯

입문 추천 시:
- Step 1 → Step 2 → Step 3를 타임라인/플로우 형태로 시각화
- 각 단계에 미니 카드 (술 이름, 맛 키워드, 가격대, 추천 음용법)
- 현재 취향 기반으로 경로가 달라짐을 시각적으로 표현

## 시각화 원칙

- CSS 변수로 다크모드/라이트모드 지원
- 모바일 반응형 (매장에서 폰으로 볼 상황 고려)
- 주류 느낌의 색상 톤 (앰버/골드 포인트, 다크 배경)
- 레이더 차트는 SVG 또는 CSS 그라디언트로 구현
- 인터랙티브 요소: 차트 호버 시 상세 수치 표시, 비교 토글

## 주의사항

- 가격 정보는 조사 시점 기준이며 실시간 변동 가능함을 안내한다
- 맛 평가는 일반적인 평가 기준이며 개인 취향에 따라 다를 수 있음을 안내한다
- 음주는 법적 음주 가능 연령 이상에게만 해당됨을 필요 시 안내한다
- 면세 한도, 주류 반입 규정 등 법적 사항을 정확히 안내한다
- 웹 검색으로 최신 가격/평가 정보를 수집하며, 불확실한 정보에는 "확인 필요"를 표기한다
- 가품 주의가 필요한 고가 주류는 정품 확인 팁도 함께 제공한다
